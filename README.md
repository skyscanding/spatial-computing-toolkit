# Spatial Computing Toolkit

GIS analysis pipelines for urban spatial computing research.

## Tools

### [QGIS Cookbook](qgis-cookbook/)
A comprehensive QGIS 3.44 operation manual — 298 native algorithms + GRASS 8.4 + SAGA + GDAL 3.11. Covers data exploration, coordinate projection, spatial filtering, geometry operations, field calculations, spatial joins, network analysis, grid statistics, GWR, export, and cartography. Everything via `qgis_process` command line.

### [GWR Pipeline](gwr-pipeline/)
ArcGIS Pro / arcpy pipeline using Geographically Weighted Regression to model facility distribution (POIs) against built-environment indicators, then predict future distribution from population and infrastructure projections. Fully automated 5-step pipeline.
